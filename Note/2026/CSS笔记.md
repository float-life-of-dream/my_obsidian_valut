# css（层叠样式表）

**定义**：用来描述HTML文档的呈现（美化内容）。

**书写位置**：title标签下方添加style双标签，style标签里面书写CSS代码

**写法**：选择器{CSS属性：属性值；} 

## CSS引入方式

##### 1.内部样式表

CSS代码写在style标签里面（学习使用）

##### 2.外部开发表

CSS代码写在单独的CSS文件中（.css）;在HTML使用  link  标签引入（开发使用）

   `<link rel="stylesheet" href="./my.css">`

##### 3.行内样式

CSS写在标签的 style 属性值里

`style="属性：属性值； 属性：属性值；"`

## 复合选择器

**定义**：有两个或多个基础选择器，通过不同的方式组合而成

**作用**：更准确，更高效的选择目标元素（标签）

### 后代选择器

后代选择器会选中**所有**后代

**写法**：父选择器 子选择器{CSS属性}，父子选择器之间用**空格**隔开

### 子代选择器

**写法：**父选择器 > 子选择器{CSS属性}，父子选择器之间用 **>** 隔开

### 并集选择器

**场景**：选中 **多组标签**设置相同的样式

选择器**写法**：选择器1，选择器2，选择器n{CSS属性}，选择器之间用  **，**隔开

### 交集选择器

**写法**：选择器1选择器2{CSS属性}，选择器之间**连写**，**没有任何符号**

### 兄弟选择器

#### 相邻兄弟选择器

div**+**p{...} ：与div**紧紧相邻**的下方第一个p（p必须处在div的**相邻下方**，若div下方不是p，则失效）

#### 通用兄弟选择器

div**~**p{...}  :选中div**以下**的**所有**p

注意：两种选择器都选择的是下方兄弟

### 属性选择器

#### 语法：

[属性名] 选中具有某个属性的元素

[属性名="值"]   选中包含某个属性，且属性值**等于**指定值的元素

[属性名^="值"] 选中包含某个属性，且属性值以指定的值**开头**的元素

[属性名$="值"] 选中包含某个属性，且属性值以指定的值**结尾**的元素

[属性名*="值"] 选中包含某个属性，且属性值**包含**指定的值的元素

## CSS三大特性

### 继承性

**子**级默认继承**父**级的**文字控制属性**

**注**：如果标签自己有样式，则生效自己的样式，不继承父级样式

#### 元素之间的空白问题

**产生原因**：行内元素，行内块元素，彼此之间的**换行**会被浏览器解析为一个**空白字符**

##### 解决：

1.去掉换行和空格（不推荐）

2.该**父级**元素设置 **font-size：0；**（使得所有子级元素（包括**换行及空格**）都继承到）再给需要**显示**的文字元素**单独设置大小**

### 层叠性

##### 特点：

1.**相同**的属性会**覆盖**：**后面**的CSS属性**覆盖前面**的CSS属性；

2.**不同**的属性会**叠加**：**不同**的CSS属性**都生效**

### 优先级

优先级也叫**权重**，当一个标签使用了**多种**选择器时，基于不同类型的选择器的**匹配规则**

#### **规则**：选择器优先级高的样式生效

#### **优先级**：

**通配符选择器<标签选择器<类选择器<id选择器<行内样式<!imporant**                        (**选中的范围越大，优先级越低**)

#### 优先级---叠加计算规则

叠加计算：如果选择器是**复合选择器**，则需要权重叠加计算

## 外边距问题

### 合并现象

##### 场景：

**垂直**排列的**兄弟**元素，**上下外边距**会合并

##### 注：两者之间的外边距会**重叠**，**取较大**的那个外边距作为两者之间的距离

##### 现象：

取两个marger 中的**较大值**生效

### 塌陷问题

##### 场景：**父子级**的标签，**子级**添加**上外边距**会产生**塌陷**问题

##### 现象：导致**父级一起**向下**移动**

#### 解决：

**取消子级的margin**，**父**级设置**padding**（为防止盒子变大同时设置 **box-sizing： border-box**）

**父**级设置 **overflow： hidden**（让浏览器找到盒子边缘）

**父**级设置 **border-top**（也会撑大盒子）

## 行内元素--内外边距问题

##### 场景：**行内**元素添加**margin和padding**，无法显示元素**垂直**位置（只改变水平方向位置）

##### 解决：

给行内元素添加**line-height** 可以改变**垂直**位置

### vw

#### 适配方案

**相对**单位

**相对视口**的尺寸计算结果

**vw**：**v**iewport **w**idth 

**1vw = 1/100 视口宽度**

**vh**：**v**iewport **h**eight

1vh = 1/100上视口高度

##### 注：一个盒子vw和vh不能混用

##### 1vmax=视口宽和视口高中大的那个的1%

##### 1vmin=视口宽和视口高中小的那个的1%

## 响应式网页

### 1.媒体查询

```
@media (媒体特性) {
  html {
    background-color: green;
  }
}
```

#### 媒体特性

**max-width**：最大宽度(网页宽度**小于等于**此宽度时生效)

**min-width**：最小宽度（网页宽度**大于等于**此宽度时生效）

#### 书写顺序

要求网页宽度 **>=** 某个宽度时，用**min-width** 并且宽度值由**小到大**写

要求网页宽度 **<=** 某个宽度时，用**max-width** 并且宽度值由**大到小**写

#### 使用：侧边栏隐藏

当网页宽度小于某个值时，给**要隐藏的区域**加**display： none;**

##### 完整写法 - 了解

```
@media 关键词 媒体类型 and (媒体特性) {CSS代码}
```

关键词 / 逻辑操作符

- and
- only
- not

**媒体类型**是用来**区分设备类型**的，如屏幕设备，打印设备，其中手机电脑平板都属于屏幕设备

![](img/431ad54445e2cde428ab3f6d6d5724bf.png)

**媒体特性**

媒体特性主要用来**描述媒体类型的具体特征**，如当前屏幕的宽度、分辨率、横屏或竖屏等

![](img/60cea3d8f9e757d937c7ca66ea0547b4.png)

## 新闻分栏式布局

column-count ：指定列数，值是数字

column-width ：指定宽度 值是长度

columns ：同时指定列宽和列数 前两者的复合属性；值没有数量和顺序要求

column-gap ：设置列边距，值是长度

column-rule-style :设置列与列之间的边框风格，与border-style 一致

column-rule-width  :设置列与列之间的边框的宽度，值是长度

column-rule-color :设置列与列之间的边框颜色

column-rule :设置列边框，前三者的复合属性

注意：以上属性均给容器设置

column-span ：指定是否跨列；值：none /all(跨所有列)（给某个标签单独设置）

如果有图片，给img标签添加（width：100%   表示图片占该栏的100%）
