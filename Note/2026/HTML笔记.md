## HTML

#### 文本标签

###### 标题标签

<h数字></h数字>      数字1~6，字体大小依次减小

特点：文字加粗； 字号减小；独占一行；

使用**限制**：h1标签在一个网页中只能用一次；

###### 字体标签

1.`<b></b>,<strong></strong>`     加粗；

2.`<sup></sup>`  我的上标

3.`<sub></sub>`  我的下标

4.`<s></s>` `<del></del>` 删除线

5.`<u></u>`  ,`<ins></ins>` 下划线

6.`<i></i>  ,<em></em>`  斜体

7.`<code></code>` ：定义计算机代码

8.`<cite></cite>`  定义引用，引证

9.`<small></small>`:定义小号字

###### 格式化标签

1.`<p></p>`   段落标签（双标签）：可以展示自然段（两段之间有空隙） p标签独占一行，自动换行；

2.

```
<br/>   换行标签 <!--单标签-->
```

3.`<hr/>`   分割线标签，换行+分割线     `<!--单标签-->`

4.`<pre></pre>`   预格式化标签：可以随意展示文本的样式和空格，但不宜使用tab键（tab键不一定表示四个空格）

#### 超链接标签

`<a></a>`

###### 作用：

点击**跳转**到其他页面；

###### 属性：

1)**href**属性可以存放图片路径,文件路径，网址，锚点；

2）**target**属性可以赋值：_self:本页面跳转； _bank:打开新的页面，原来页面与要跳转页面同时存在；[name]:在指定名字的标签上打开，一般配合 iframe 标签使用

注：在**未知**即将要跳转的地址时，先将href定为**#**（此时页面不会完成跳转）

###### 属性：（属性不分先后位置，且以空格分开）

##### 音频标签

`<audio src="音频" controls></audio>` （不加控制面板无法显示）

###### 属性：

**src**(必须属性)：音频

**controls**：显示音频**控制面板** （必须书写）

**loop**：循环播放

**autoplay**:自动播放（浏览器**禁用**）

**注**：在H5中，如果属性名与属性值相同，可以简写为一个单词

##### 视频标签

`<video src="视频" controls></video>`  （不加控制面板无法显示）

###### 属性：

src;

controls;

loop;

**muted**:静音播放；

autoplay:自动播放（浏览器支持在静音状态自动播放）

注：想要自动播放，就要加muted标签，否则无法自动播放

#### 列表标签

##### 无序列表

无序列表是最常用的列表，ul是定义无序列表的标签，li是具体的列表项

**注**：ul标签里面只能包含li标签

type属性：circle  disk  square

格式：

```
<ul type="circle/disk/square">
    <li>列表内容</li>
    <li>列表内容2</li>
    </ul>
```

##### 有序列表

有序列表是可以在列表项前标记序号，ol是定义有序列表的标签，li是具体的列表项

**注**：ol标签里面只能包含li标签,想放其他的东西在li里面嵌套

type属性：1  i  I  A  a

格式：

```
<ol type="type属性">
    <li>列表内容</li>
    <li>列表内容2</li>
    </ol>
```

##### 自定义列表

dl是定义自定义列表的标签，dt可以设置标题，dd是具体的列表项

**注**：dl标签里面只能包含dt和dd标签

格式：

```
<dl>
<dt>标题</dt>
<dd>内容</dd>
<dd>内容</dd>
</dl>
```

**注**：内容会相对标题**自动缩进**

#### 表格标签

##### `<table></table>`

table嵌套tr,tr嵌套td/th。

表格标签用于编制表格，**table**用于定义**表格**，**tr**定义表格**行**，**td**定义**单元格**（内容单元格），**th**定义单元格的同时默认单元格中的内容居中加粗（**表头**单元格）

td/th属性：大多数属性可以在样式表中定义（**colspan 跨列**属性；**rowspan 跨行**属性）

格式：

```
<table border="1">
<tr>
<td conlspan="2">单元格内容1</td>   <!--表示内容1占据横向的两个单元格-->
<td rowspan="2">单元格内容3</td>    <!--表示内容3占据纵向的两个单元格-->
</tr>
<tr>
<td>单元格内容1</td>
<td>单元格内容2</td>
</tr>
</table>
```

#### form表单标签

表单标签侧重于与后端交互数据（用户输入信息）

主要以 input 文本标签为主，辅之以name属性，通过form表单向后端发送数据

除了input标签外还有大文本标签 textarea ；下拉菜单标签 select ;

##### 1.input标签（单标签）

###### 属性：

1.**type**:text（文本框，用于输入单行文本）,password（密码框，输入内容以点的形式显示）count,date,botton,radio（单选框）,checkbox(多选框)，file（上传文件，一次只能上传一个文件）hidden等；

2.**placeholder**:文本框，密码框中用来提示用户输入的**提示信息**；

3.**name**:name设置的值必须要和后端函数设置的参数名一致；radio单选时，加name属性(name相同的radio选项，为多选一)；

4.**checked**：radio单选时，加name属性(name**相同**的radio选项，为多选一)，在哪个里面加checked表示默认哪个；**checkbox**多选时，在默认选项内容的input标签中加checked属性；

5**.multiple**:想要选择**多个文件**在type="file"后加multiple属性

6.**value**:显示在页面上的值，除了 type='hidden' 的除外，因为 type='hidden' 属于隐藏标签；

7.**required**:未输入内容时**阻止表单提交**  **表单验证时使用**

##### 2.下拉菜单

select嵌套option：**select**是下来菜单的**整体**，**option**是下拉菜单的每一**项**

**默认**选项：在想要默认选择的内容的option标签中加selected（不加selected时默认**第一个**option的内容）

##### 3.文本域

作用：**多行输入**文本的表单控件

**标签**`<textarea></textarea>`

**属性**：

name,id,cols,rows

**注**：文本框右下角有拖拽功能，未来要**禁用**；尺寸用**css设置尺寸**，而非html中的textarea标签设置

##### 4.lable标签（双标签）

**作用**：网页中，某个标签的说明文本

用lable标签绑定文字与表单控件的关系，**增大**表单控件的**点击范围**

**写法**：

1.lable只包裹内容，不包含表单控件

设置lable标签的**for**属性值和表单控件的**id**属性值相同

**2.**用lable标签将input标签**全部包裹**，实现点击范围扩大（即点字也能选中）

##### 5.按钮—button(双标签)

###### type属性:

1.**submit**:提交按钮，点击后可以提交数据到后台（**默认功能**，可省略）

2.**reset**:重置按钮，点击后将表单控件恢复默认值

3.**button**：普通按钮，默认没有功能，一般配合javaScript使用

**注**：要配合form标签使用

#### `iframe标签`

iframe标签用于在一个页面中**嵌入**另一个页面；

配合a标签使用时，可以用a标签指定iframe指向的页面；

**使用**：

```
<a href="http://www.baidu.com" target="iframe-test">显示百度</a>
<iframe name='iframe-test' width='1000' height='1000' src="网址1"></iframe>
（iframe标签默认 src 指向的网址1，通过a标签改变页面）
```

#### div&span标签

1）html文档中最重要的容器标签；

2）div是**块**状容器，独占一行(h,p标签也独占一行)，主要用于搭建网页布局，页面整体布局就靠它；

3）span是**行**级容器（不换行），其大小自适应与包裹的对象；

使用：

```
<div>
<span>Hello </span>
<span>World!</span>
</div>
```
