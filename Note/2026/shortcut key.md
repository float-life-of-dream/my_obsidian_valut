# 快捷键

## 系统

- Win+Ctrl+v 音量

##  录屏

- Windows 徽标键 + G：打开游戏栏
- Windows 徽标键+ Alt + G：录制最后 30 秒 (可以更改 游戏栏 > 设置 )
- Windows 徽标键 + Alt + R：开始/停止录制
- Windows 徽标键 + Alt + B：打开或关闭 HDR
-  Windows 徽标键‌+ Shift+ S

## 切换窗口

- Alt+Tab 
- Win+Tab
