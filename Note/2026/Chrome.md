#### 打开标签页组的菜单

**在 Windows 上：**

1. 在 Windows 计算机上打开 Chrome。
2. 若要选择标签页，请按 **F6** 直到您的标签页获得焦点。
3. 若要将焦点移到特定标签页，请按 **Tab** 键向前移动，或按 **Shift + Tab** 向后移动。
4. 若要打开标签页分组菜单，请按 **菜单** 。
5. 若要在菜单中进行选择，请执行以下操作：
	- 若要向上或向下移动，请按向上键 或向下键 。
	- 若要向左或向右移动，请按向左键 或向右键 。
	- 若要选择菜单中的某个选项，请按 **Enter** 键。

**在 Chromebook：**

1. 在 Chromebook 上，打开 Chrome。
2. 若要选择标签页，请按 **Ctrl + Tab** 。
3. 若要将焦点移到特定标签页，请按 **Ctrl +** 后退键 或 **Ctrl +** 前进键 。
4. 若要打开标签页分组菜单，请执行以下操作：
	- 按搜索键 **\+ Shift +** “调高音量”图标 。
	- 您也可按启动器键 **\+ Shift +** “调高音量”图标 。
5. 若要在菜单中进行选择，请执行以下操作：
	- 若要向上或向下移动，请按向上键 或向下键 。
	- 若要向左或向右移动，请按向左键 或向右键 。
	- 若要选择菜单中的某个选项，请按 **Enter** 键。

#### 移动、收起和展开标签页分组

您可收起标签页分组，使系统仅显示其名称或彩色圆圈。您还可展开标签页分组，使系统显示其中的所有标签页。

若要收起或展开标签页分组，请执行以下操作：

1. 将焦点移到标签页分组标题。
2. 按 **空格键** 或 **Enter** 键。

**提示：** 当标签页分组处于收起状态时，您只能选择该组，无法选择该组中的具体标签页。

若要移动标签页或标签页分组，请执行以下操作：

1. 将焦点移到该标签页或标签页分组。
2. 按 **Ctrl +** 向左键 或 **Ctrl +** 向右键 。
